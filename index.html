Projektit

A Parametric 3-Band Audio Equalizer
2025

A cross-browser audio equalizer extension for Chromium and Firefox browsers using the Web Audio API. Built with HTML, JavaScript, and CSS, the extension offers customizable frequency bands, gain and Q factor controls, as well as preamp and master volume adjustment. Features include a real-time frequency response graph, a live audio spectrum analyzer with bar visualization, per-tab state management, and a collection of preconfigured presets with a modern user interface.

JavaScript (ES6+), HTML5, CSS3, Web Audio API, Chrome Extension API
