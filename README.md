## Portfolio Website

A lightweight, modern personal website with an easy-to-navigate design

## ✨ Features

* 🌊 **Ocean-inspired theme**
  * Idea was to create modern and unique "in-depth" portfolio website

* 📱 **Responsive design**

  * Works across mobile, tablet, and desktop

* ⚡ **Lightweight & fast**

  * No frameworks (except Tailwind CDN)

---

## 🛠️ Tech Stack

* **HTML5**
* **CSS3**
* **JavaScript (ES6+)**
* **Tailwind CSS (CDN)**
